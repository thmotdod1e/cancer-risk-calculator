# Cancer Risk Calculator

A simple command-line tool that calculates an estimated lung cancer risk score based on smoking-related variables, with special emphasis on the "age of first cigarette consumption" as a primary predictor.

This tool is intended for educational purposes and could be integrated into smoking cessation program materials.
